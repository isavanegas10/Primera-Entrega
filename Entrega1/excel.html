Fórmula

PROMEDIO(rango;)

¿Para qué?

Es utilizado para calcular el promedio de ciertas celdas

Pasos

  1. Damos click en la celda donde queremos visualizar el resultado
  2. Se escribe la formula =PROMEDIO
  3. Seleccionamos las celdas para calcular el promedio
  4. Damos ENTER y el resultado aparecera

Fórmula

=CONTAR.SI (rango;criterio)

¿Para qué?

Sirve para contar aquellas celdas que contienen numeros o valores dentro de cierto rango, tambien puede ser usado como herramienta de busqueda para una informacion especifica

Pasos

  1. Damos click en la celda donde queremos visualizar el resultado
  2. Escribimos =CONTAR.SI
  3. Seleccionamos las celdas y/o el condicional deseado
  4. Damos ENTER y el resultado aparecera
